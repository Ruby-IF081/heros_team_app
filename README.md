## To run application you need:
* Ruby `v.2.4.1`
* Redis server
* PostgreSQL

## Installation

* Run `bundle`
* Run `rails db:migrate`
* Run `rails db:seed`
* Start `redis-server`
* In project folder run `bundle exec sidekiq`
* Run `rails s`
